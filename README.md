Config that I use regularly for computers. 

# Content

## Terminal Functions

### docs

Enables quicker control over my documents.

Includes functions:
- `docs`
- `docs_cd`
- `docs_pull`
- `docs_push`

### saved-pwd

Enables saving a directory for quicker access between terminals.

Includes functions:
- `get_pwd`
- `save_pwd`
- `swap_pwd`
- `last_pwd`
- `last_saved_pwd`
