# Summary

Config that I use regularly for computers. 

# Content

## Dotfiles

### Bash

Configuration for my bash shell, includes `.bashrc`

### Neovim

Configuration for my neovim code editor, stored in `~/.config/nvim`

### Tmux

Dotfiles for tmux. Includes ~/.tmux.conf

## Terminal Functions

### docs

Enables quicker control over my documents.

Includes functions:
- `docs`
- `docs_cd`
- `docs_pull`
- `docs_push`

### saved-pwd

Enables saving a directory for quicker access between terminals.

Includes functions:
- `get_pwd`
- `save_pwd`
- `swap_pwd`
- `last_pwd`
- `last_saved_pwd`
